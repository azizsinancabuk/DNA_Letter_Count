# DNA Letter Count

A Python project to count DNA nucleotides and draw a bar chart.

## Description
This project reads a DNA sequence, counts the occurrences of each nucleotide (A, T, G, C), and visualizes the results using a bar chart with matplotlib.

## Requirements
- Python 3.x
- matplotlib library

## How to Run
1. Make sure Python 3.x is installed.
2. Install matplotlib if not already installed:

pip install matplotlib
3. Run the `main.py` file:
python main.py
